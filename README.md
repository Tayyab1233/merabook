# MeraBook (MyBook)

A lightweight Facebook clone built to demonstrate core React concepts. Users can register, log in, create text and image posts, and view a global feed of posts from all users. 

## Features
* **User Authentication:** Sign up and log in functionality with secure route protection.
* **Global Feed:** View all posts from all registered users in chronological order.
* **Create Posts:** Upload text and attach images (converted to Base64).
* **Edit Posts:** Users can securely edit their own previous posts.
* **Responsive Design:** Clean and interactive user interface.

## Tech Stack
* React
* Vite
* React Router DOM
* CSS (Custom Styling)

## Setup Instructions
To run this project locally on your machine:
1. Clone the repository.
2. Run `npm install` to install all dependencies.
3. Run `npm run dev` to start the local development server.
